📝 Simple Text Editor

📌 Description

This is a basic text editor built using Python and Tkinter.
It allows users to create, open, edit, and save text files.

🚀 Features
Create new file
Open text file
Save file
Simple user interface

🛠️ Technology
Python
Tkinter

▶️ How to Run
1. Save the code in a file named main.py
2. Run the file using:
python main.py
